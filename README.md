# brickhack-mobile-ios

## Dependencies
We're using CocoaPods as the dependency manager.

Installing CocoPods:

`$ sudo gem install cocoapods`

Installing dependencies:

`$ pod install`

Verify Apollo was installed correctly:

`$ pod try Apollo`
